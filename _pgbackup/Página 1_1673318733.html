El Mol

En la química la obteción de productos a partir de reactivos se expresa a través de ecuaciones químicas. En las ecuaciones químicas se mezclan reactivos para obtener productos, por ejemplo:

NaOH + HCl ---------> NaCl + H2O

Para ello es necesario saber que cantidad de reactivos debemos combinar para obtener una cierta cantidad de productos. Una forma sería considerar mezclar una molécula de hidróxido de sodio con una molécula de ácido clorhídrico, pero no podemos hacerlo debido a que son demasiado pequeñas las moléculas (microscópicas) como para manejarlas. Lo mismo sucede si tomamos el peso de una molécula, ya que el peso molecular se da en uma (unidad de masa atómica), no existe una balanza que pese estas unidades de masa. ¿Entonces cómo hacerlo? Aquí es donde entra la estequiometria.

Estequiometria. Es la rama de la química que se encarga del cálculo de las relaciones cuantitativas entre los reactivos y productos en el transcurso de una reacción química.

Para realizar la determinación de cuanto material debemos usar para una determinada reacción química, tenemos la unidad fundamental que nos indica la "cantidad de sustancia" conocida como mol.


Mol
Un mol es la cantidad de sustancia de un sistema que contiene tantas entidades elementales como átomos hay en 0.012 kilogramos de carbono 12, su símbolo es "mol".

Para entenderlo mejor veamos lo siguiente:

A 2 átomos de un elemento le llamamos bina.
A 3 átomos de un elemento le llamamos terna.
A 10 átomos de un elemento le llamamos decena.
A 12 atomos de un elemento le llamamos docena.
A 6.022 x 1023 átomos de un elemento le llamamos mol.

Es decir:

1 mol de Au contiene 6.022 x 1023 átomos de Au,
1 mol de Na contiene 6.022 x 1023 átomos de Na,
1 mol de H2 contiene 6.022 x 1023 moléculas de H2,
1 mol de H2O contiene 6.022 x 1023 moléculas de H2O.

Ahora bien,

1 mol de cualquier sustancia contiene 6.022 x 1023 partículas (átomos, moléculas, iones).
0.5 mol de cualquier sustancia contiene 0.5 x (6.022 x 1023) = 3.011 x 1023 partículas (átomos, moléculas, iones),
0.2 mol de cualquier sustancia contiene 0.2 x (6.022 x 1023) = 1.2044 x 1023 partículas,
2 mol de cualquier sustancia contiene 2 x (6.022 x 1023) = 12.044 x 1023 partículas

Finalmente, a la relación de partículas contenidas en 1 mol, le conocemos como numero de Avogadro.

NA = 6.022 x 1023 partículas/mol


Masa molar
La masa molar de cualquier sustancia es la cantidad de masa contenida en 1 mol de esa sustancia, y es numéricamente igual a su masa molecular (atómica) pero expresada en gramos.

De acuerdo a la definición de masa molar, la masa molar dependerá de la sustancia en particular de la que estemos hablando.

Así entonces tenemos que:

1 mol de Na (masa atómica = 22.98 uma) tiene una masa de 22.98 gr.
La masa molar del Na será de 22.98 gr/mol.

1 mol de H2 (masa molecular = 2 x 1 uma = 2 uma) tiene una masa de 2 gr.
La masa molar del H2 es de 2 gr/mol.

1 mol de HNO3 (masa molecular = 1 + 14 + 3 x 16 = 63 uma) tiene una masa de 63 gr.
La masa molar del HNO3 es de 63 gr/mol.

Ahora bien

0.5 mol de HNO3 tendrá 0.5 x 63 gr/mol = 31.5 gr,
0.2 mol de HNO3 tendrá 0.2 x 63 gr/mol = 12.6 gr,
3 mol de HNO3 tendrá 3 x 63 gr/mol = 189 gr.

De lo anterior podemos definir la siguiente ecuación

\(\displaystyle M=\frac{m}{n}\)

Siendo:
   M - La masa molar, en gr/mol
   m - La masa, en gramos
   n - La cantidad de sustancia, en moles 


Volumen molar
El volumen molar de cualquier sustancia es el volumen ocupado por 1 mol de esa sustancia.

Para el caso particular del estado gaseoso, 1 mol de cualquier gas ocupara el mismo volumen, si están a la misma presión y temperatura. Este volumen será de 22.414 m3 cuando se encuentran a una presión de 101,325 Pa y 273.15°K, dicho volumen se conoce como "volumen molar normal". En condiciones ambientales de 101,325 Pa y 298.15 °K, el volumen molar es de 24.466 m3.

Para sustancias en estado sólido y líquido tenemos el volumen molar como

\(\displaystyle M_m=\frac{V}{n}\) 

Siendo:
   Vm - Volumen molar
   V - Volumen
   n - Numero de moles.

a) 103 x 104 = 103+4 = 107 = 1 x 107

b) (1 x 103) x (1 x 102)= 1 x 1 x 103+2 = 1 x 105

c) (2 x 104) x (3 x 102) = 2 x 3 x 104+2 = 6 x 106

d) (5 x 102) x (4 x 105) = 5 x 4 x 102+5 = 20 x 107 = 2 x 108

e) (4 x 106) x (2 x 10-2) = 4 x 2 x 106+(-2) = 8 x 104

f) (6 x 10-3) x (5 x 10-4) = 6 x 5 x 10(-3)+(-4) = 30 x 10-7 = 3 x 10-6

División de potencias.
Se le resta al exponente del numerador el exponente del denominador.


   \(\displaystyle \frac{1}{{10}^{4}}={10}^{-4}=1\times {10}^{-4} \)

   \(\displaystyle \frac{1}{{10}^{-2}}={10}^{2}=1\times {10}^{2} \)

   \(\displaystyle \frac{6}{3\times {10}^{4}}=\frac{6}{3}\times \frac{1}{{10}^{4}}=2\times {10}^{-4} \)

   \(\displaystyle \frac{8\times {10}^{2}}{4}=\frac{8}{4}\times {10}^{2}=2\times {10}^{2} \)

   \(\displaystyle \frac{12\times {10}^{6}}{6\times {10}^{4}}= \frac{12}{6}\times \frac{{10}^{6}}{{10}^{4}}=2\times {10}^{6}\times {10}^{-4}=2\times {10}^{2} \)

   \(\displaystyle \frac{25\times {10}^{-2}}{5\times {10}^{4}}=\frac{25}{5}\times \frac{{10}^{-2}}{{10}^{4}}=5\times {10}^{-2}\times {10}^{-4}=5\times {10}^{-6} \)

   \(\displaystyle \frac{45\times {10}^{-8}}{15\times {10}^{-3}}=\frac{45}{15}\times \frac{{10}^{-8}}{{10}^{-3}}=3\times {10}^{-8}\times {10}^{3}=3\times {10}^{-5} \)

   \(\displaystyle \frac{5\times {10}^{7}}{2\times {10}^{-4}}=\frac{5}{2}\times \frac{{10}^{7}}{{10}^{-4}}=2.5\times {10}^{7}\times {10}^{4}=2.5\times {10}^{11} \)

Suma y resta de potencias.
Para efectuar estas dos operaciones los exponentes deben ser iguales. En caso contrario tenemos que igualarlos, ya sea aumentar uno o disminuir el otro.

a) (2 x 103) + (3 x 103) = 5 x 103

b) (8 x 108) + (0.5 x 108) = (8.5 x 108)

c) (7 x 1012) + (9 x 1012) = (16 x 1012)

d) (15 x 1024) - (12 x 1024) = (3 x 1024)

e) (3 x 1025) - (2 x 1025) = (1 x 1025) 

f ) (18 x 1026) - (20 x 1026) = (22 x 1026)

g) (9.5 x 104) - (3 x 105) = ¡no pueden sumarse como en los casos anteriores!

En este último caso debemos igualar sus exponentes. Para ello, aumentamos el exponente menor o disminuimos el mayor y el resultado será el mismo. Si aumentamos el menor, tenemos que:

   (9.5 x 104) = (0.95 x 105)

donde:

   (0.95 x 105) - (3 x 105) = 3.95 x 105

Si disminuimos el exponente mayor, tenemos que:

   (3 x 105) = (30 x 104)

donde:

   (9.5 x 104) - (30 x 104) = -21.5 x 104 = -2.15 x 106

h) (4.5 x 108) - (2 x 1010) al disminuir el exponente mayor, tenemos:

   (4.5 x 108) - (200 x 108) = 204.5 x 108

al aumentar el menor, tenemos:

   (0.045 x 1010) - (2 x 1010) = -1.955 x 1010

Elevación de un exponente a otro exponente. 
Se multiplican los exponentes.

a)  (105)2 = 105x2 = 1010 = 1 x 1010

b) (10-4)3 = 10-4x3 = 10-12 = 1 x 10-12

c) (5 x 102)2 = 52 x 102x2 = 25 x 104

d) (6 x 103)2 = 62 x 103x2 = 36 x 106

e) (2 x 105)4 = 24 x 105x4 = 16 x 1020  

Raíz cuadrada.
Cuando el exponente es par se procede a obtener la raíz cuadrada directamente.
 
Cuando el exponente es impar, debe convertirse a exponente par, para no obtener un exponente fraccionario como resultado.

   \(\displaystyle \sqrt{{10}^{4}}={10}^{\left(4\div 2\right)}={10}^{2} \)

   \(\displaystyle \sqrt{{10}^{8}}={10}^{\left(8/2\right)}={10}^{4} \)

   \(\displaystyle \sqrt{{10}^{2}}={10}^{\left(2/2\right)}=10 \)

   \(\displaystyle \sqrt{9\times {10}^{6}}=\sqrt{9}\times {10}^{\left(6/2\right)}=3\times {10}^{3} \)

   \(\displaystyle \sqrt{25\times {10}^{12}}=\sqrt{25}\times {10}^{\left(12/2\right)}=5\times {10}^{6} \)

   \(\displaystyle \sqrt{64\times {10}^{7}}=\sqrt{6.4\times {10}^{8}}=\sqrt{6.4}\times {10}^{\left(8/2\right)}=2.53\times {10}^{4} \)

   \(\displaystyle \sqrt{90\times {10}^{-5}}=\sqrt{9.0\times {10}^{-4}}=\sqrt{9}\times {10}^{\left(-4/2\right)}=3\times {10}^{-2} \)

   \(\displaystyle \sqrt{5\times {10}^{-9}}=\sqrt{50\times {10}^{-10}}=\sqrt{50}\times {10}^{\left(-10/2\right)}=7.071\times {10}^{-5} \)
